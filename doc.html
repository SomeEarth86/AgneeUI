Introduction
Agnee UI is CSS library with built-in components which helps you to ship your project faster.
Installation
Copy the following stylesheet in your head tag. 
                      
<link rel="stylesheet" href="https://agnee-ui.netlify.app/css/components.css">
Starter Template
                      
<html lang="en">
    <head>
        <!-- required metatags -->
    <meta charset="UTF-8">
    <meta http-equiv="X-UA-Compatible" content="IE=edge">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
                        
    <link rel="stylesheet" href="https://agnee-ui.netlify.app/css/component.css ">
                        
    <title>Title</title>

</head>

<body>
    
    <h1>Agnee UI</h1>

</body>

</html>
