Avatar
Avatar are profile picture use to display ourselves on various social media.
Profile Avatar
Profile Avatar can be applied to your project using the class 'avatar' inside the image tag. 
                    
<img class="avatar" src="https://rb.gy/mfpksz" alt="avatar-image" >
<img class="avatar" src="https://rb.gy/quesxa" alt="avatar-image" >
<img class="avatar" src="https://rb.gy/0v802a" alt="avatar-image" >
Size Avatar
Size avatar basically refers to the various size that user wish to implement image according to their usage. We support 3 different sized namely small, medium and large using classes 'sm | md | lg' respectively. 
                    
<img class="avatar lg" src="https://rb.gy/mfpksz" alt="avatar" />
<img class="avatar md" src="https://rb.gy/mfpksz" alt="avatar" />
<img class="avatar sm" src="https://rb.gy/mfpksz" alt="avatar" />
